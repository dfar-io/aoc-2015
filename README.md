Advent of Code Solutions

Potential language choices:
* 2015 - C#
* 2016 - Python 3
* 2017 - Rust
* 2018 - TypeScript (inside Angular?)
* 2019 - Kotlin
* 2020 - F#
* 2021 - Go
* 2022 - Haskell
* 2023 - Ruby
* 2024 - PHP

## Getting Started

To run a problem, provide the input in the `input` directory (example: `01.txt`), then run, providing the problem number
as an argument.
